# Phase 3 - Security and Detection

## Objective

Establish SIEM-lite capability for home/lab with actionable detections and controlled response.

## Outcomes

- Practical detection coverage for home threat model
- Correlated events across router, DNS, and hosts
- Security operations dashboard and tuning loop

## Scope

- Detection content: auth failures, DNS anomalies, suspicious outbound trends
- Event enrichment and tagging for incident context
- Response workflow design with approval gates

## Deliverables

- Detection pack v1 (15+ detections)
- Security dashboard with severity and trend views
- Response runbooks with AWX template mappings

## Exit Criteria

- Detection quality metrics in place
- High-severity detections linked to explicit response paths
